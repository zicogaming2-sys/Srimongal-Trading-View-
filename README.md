# Srimongal Trading View

Srimongal Trading View is a powerful and clean crypto trading chart platform for real-time market analysis.  
View live Bitcoin, Ethereum, and forex charts with advanced tools, multiple timeframes, and a professional dark interface — all without ads or branding.

---

## Features
- Fullscreen crypto chart
- Live market updates
- Dark, modern UI
- Works directly in browser (no login needed)

---

## Run Locally
1. Open `index.html` in your browser.

## Deploy to GitHub Pages
1. Push the repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Branch**, select `main` and **Folder: root**.
4. Click **Save** — your live website link will appear automatically.
